## General info

## Technologies
Project is developed with:
Java 1.8
org.jsoup 1.9.2

## Setup
To run this project, execute below commands:
$cd project root folder
$mvn compile
$mvn exec:java -Dexec.mainClass=com.program.exercise.App -DLOG_PATH=value -Dexec.cleanupDaemonThreads=false
